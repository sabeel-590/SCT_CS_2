## Project Overview

The Image Encryption Tool is a Python-based GUI application developed as part of the SkillCraft Technology Cyber Security Internship.

This project demonstrates image encryption and decryption using pixel manipulation techniques. The application modifies pixel values using a mathematical operation and restores the original image during decryption using the same key.

The tool provides a user-friendly graphical interface that allows users to securely encrypt and decrypt image files.

## Features

Select image from local system

Encrypt image using pixel manipulation

Decrypt encrypted image

User-defined encryption key

Interactive Graphical User Interface (GUI)

Dark-themed professional design

Supports JPG, JPEG, PNG and BMP images

Easy-to-use interface

Secure image transformation

## How the Encryption Works

Encryption Formula:

Pixel = (Pixel + Key) % 256

Decryption Formula:

Pixel = (Pixel - Key) % 256

## Example
